# Quora Posts App 📝

A simple Quora-style web app to create, edit, delete, and view posts.

## 🔧 Tech Stack
- Node.js + Express.js
- EJS Templates
- CSS Animations + Theme Switcher + Dark Mode
- FontAwesome Icons

## 🌟 Features
- Create/Edit/Delete posts
- Dark mode toggle
- Theme dropdown (Blue, Green, Pink)
- Toast notifications
- Smooth page transitions

## 🚀 How to Run
```bash
npm install
node index.js
